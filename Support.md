# Shorts and Feed Blocker

## Overview
Shorts and Feed Blocker is a browser extension designed to help you stay focused by removing distractions from YouTube.  
It allows you to customize your viewing experience by blocking Shorts, recommendations, and comments.

The goal is to create a cleaner, more intentional way to use YouTube.

---

## Features
- Block YouTube Shorts from feed and search  
- Hide recommended / suggested videos  
- Hide comments section  
- Simple toggle-based controls  
- Enable or disable each feature individually
